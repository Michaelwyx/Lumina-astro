远程成像(remote astrophotography)指通过互联网调用架设在优质台址(暗夜、晴夜率高、视宁度好)上的望远镜进行拍摄,或直接下载这些望远镜已采集的数据。它使观测者无需拥有设备、无需前往现场,即可获取深空(deep-sky)目标的原始数据。本页对主流平台进行分类对比,逐一说明其类别、设备、半球与台址、计费模式、适用人群与优缺点。

在对比之前需要先做类别区分:相关服务并非同一类产品,解决的是不同环节的问题。

| 类别 | 含义 | 代表平台 |
| --- | --- | --- |
| 图库 / 社区 / 数据库 | 托管、展示、检索图片;查询器材与采集参数;**不出租望远镜** | AstroBin |
| 远程望远镜网络 | 在多台址布有多台镜,**实时控制**拍摄并下载数据 | iTelescope、Slooh、ChileScope |
| 远程望远镜(单 / 少设备) | 围绕一两套设备提供**按夜或按请求**的拍摄服务 | SkyShare Astro |
| 数据集 + 远程混合 | 既可请求拍摄,也可**下载平台已拍好的校准数据集** | Telescope Live |
| 远程托管(hosting) | 把**你自己的设备**架到对方台址代为运营 | Deep Sky Chile、Insight Observatory(含远程拍摄) |

<Aside type="note" title="范围说明">
本页只讨论「远程成像 / 图库」类服务。Sky-Watcher、ZWO 等是望远镜与相机的**硬件厂商**,与「自建后院或便携设备」相关,不属于远程平台范畴,故不在对比之列(其产品可能作为某些远程台址上的设备出现,例如 SkyShare 使用的 Sky-Watcher Esprit 100ED)。
</Aside>

## 远程成像的共性概念

在比较具体平台前,先明确几个贯穿全篇的术语与计费要素。

- **机时计费**:多数网络平台按「望远镜使用时长」收费,常用单位为**点数(points)**或**小时**。点数与美元的换算因平台而异(例如 ChileScope 约 1 点 = 1 美元),实际消耗取决于所选望远镜、曝光时长与帧数。
- **订阅 + 机时**:许多平台采用「月订阅(享折扣机时与功能)+ 按使用扣点」的双层结构,订阅档越高,单位机时折扣越大、附带点数越多。
- **One-Click / 数据集**:部分平台提供「一键」获取平台**已采集**的校准 / 未校准数据,无需自己排程拍摄,适合直接进入[后期叠加](/astrophotography/processing/stacking/)练习。
- **交付格式**:科研级与进阶平台通常交付 **16 位 FITS** 原始帧,并附**定标帧(calibration frames)**(暗场 dark、平场 flat、偏置 bias);面向新手 / 教育的平台可能只交付预览图(JPEG / PNG)。
- **半球与台址**:南半球台址(智利、澳大利亚)可拍北半球看不到的南天目标(如大、小麦哲伦云、船底座星云);**Bortle 等级**(光害分级,1 级最暗)与**视宁度(seeing,角秒)**直接决定可达到的暗弱目标与分辨率。

<Aside type="caution" title="常见误区">
- **AstroBin 不能用来拍照**:它是图库与器材数据库,只能存图、查图、查参数。
- **价格与配置会变**:下文数值为撰写时各平台公开信息的量级参考,实际以官网为准。
- **机时费会叠加**:网络平台「订阅 + 机时」结构下,长曝光大项目的总花费可能远超月订阅本身。
</Aside>

## AstroBin

- **类别**:天文摄影图片托管平台 + 社区 + 器材 / 天体数据库 + 二手市场。**不出租望远镜,本身不能拍照。**
- **设备 / 台址**:无(不运营望远镜)。但收录 20 余家远程台址作为合作伙伴,可在站内检索这些设备拍出的样张。
- **核心功能**:图片托管、论坛、器材数据库(按型号检索镜 / 相机 / 滤镜)、自动板解(plate-solving)与天体识别、按多维度的图片检索。
- **计费(年订阅)**:

| 档位 | 定位 | 关键能力 |
| --- | --- | --- |
| Free | 试用 / 浏览 | 上传额度有限(约 10 张),可浏览全站与论坛 |
| Premium | 满足多数人 | 充足上传额度,完整托管与展示 |
| Ultimate | 最常购买 | PixInsight 驱动的高级板解(更大星表、识别行星与小行星、生成详细 finding chart),解锁全部检索过滤器:赤经赤纬(RA/Dec)、焦距、口径、相机像元、Bortle、月相、采集月份等 |

- **适合人群**:需要**存放 / 展示成果**、**查询某天体他人怎么拍**、**核对器材与采集参数**的人。可与任意远程拍摄平台配合,作为参数参考与作品归档。
- **优点**:器材与天体检索体系完善;免费档即可浏览与少量上传;板解结果可直接关联到[星表](/reference/catalog/)中的天体。
- **缺点**:不提供任何拍摄能力;高级检索与高级板解需 Ultimate。

建议先用免费账号熟悉目标检索与「采集参数(acquisition)」页面,再决定是否升级。

## iTelescope

- **类别**:老牌远程望远镜租用**网络**,自 2006 年运营。
- **设备 / 台址**:约 20 余台联网望远镜,分布于**美国(新墨西哥)、澳大利亚(Siding Spring)、西班牙 / 拉帕尔马**等多台址,**横跨南北半球**;焦距从广角折射镜到长焦反射镜不等,可打小星系 / 行星(长焦)与大星云(广角)。
- **计费(订阅 + 机时)**:月订阅(28 天周期,约 $39.95 起,高档约 $160)**+ 按望远镜使用时长扣点 / 计时**。订阅会员另享每次会话约 **30 分钟免费**机时;不同望远镜按档位有不同的「点 / 小时」费率,顶级大镜更贵。
- **使用模式**:**实时控制**(设定坐标、选滤镜、设曝光、排程),拍完数分钟即可下载;也提供「一键」图片服务降低门槛。
- **适用对象**:深空 LRGB / 窄带成像,以及**科学用途**——测光(photometry)、系外行星、彗星、小行星 / 近地天体(NEO)、超新星、变星等。
- **优点**:设备多样、双半球可拍南天专属目标;实时控制、出图快;社区与教程完善;支持科研。
- **缺点**:订阅与机时叠加,大项目成本高;界面偏传统,有一定学习曲线。

**适合**:想要目标多样性、双半球覆盖、实时控制、未来涉足科学观测的人。

## SkyShare Astro

- **类别**:较新的远程租用服务,围绕**单套高端广角设备**,按夜 / 按请求提供拍摄。
- **设备 / 台址**:**Sky-Watcher Esprit 100ED**(550mm,f/5.5 APO 三片复消色差折射镜)+ 多位滤镜轮(LRGB 与窄带 Hα/OIII/SII),架设于智利 **El Sauce 天文台**附近的 **Bortle 1–2 极暗天空**下。
- **计费(按夜包干,无点数)**:按晚一口价(公开档位约 Student $69、Discovery 首单 $79、Standard $99 量级,多夜包更省)。
- **交付**:约 **24 小时内**给出下载链接,含 **16 位 FITS 原始帧 + 定标帧 + 处理后的 JPEG 预览**。
- **天气保证**:若天气导致无法采集,**改期或退款**,阴天不计费。
- **优点**:流程简单、价格可预测;550mm 广角适合**大视场明亮星云**(M42、M31、北美星云、心脏 / 灵魂星云);天气保证降低无效拍摄风险;直接交付可处理的 FITS。
- **缺点**:**设备单一**(仅一套 550mm 广角)——不适合小星系、行星与小目标;一晚通常只服务一个目标;平台较新,口碑仍在积累。

**适合**:第一次尝试远程深空、希望减少操作复杂度、主要拍广角大星云的人。

## Telescope Live

- **类别**:远程机器人望远镜平台,兼具**实时 / 排程拍摄**与**可下载数据集**。
- **设备 / 台址**:全球多台址(**智利、西班牙、澳大利亚**),望远镜口径从约 **100mm 折射镜到 1 米(1000mm)级反射镜**不等,覆盖广角到长焦需求,横跨南北半球。
- **计费(订阅 + 点数)**:分级月订阅,各档每月发放不同点数,例如 Bronze(约 5 点)/ Silver(约 20 点)/ Gold(约 50 点)/ Platinum(约 100 点)/ Diamond(约 250 点);价格由最低档约 €6/月 到最高档约 €299/月。提供约四周的免费试用期。
- **数据获取方式**:
  - **One-Click Observations**:浏览并下载平台过去拍摄的目标数据集(可选原始或已校准),按订阅档位决定可下载范围。
  - **高级请求(Advanced Requests)**:自定义目标、滤镜、曝光,排程到指定望远镜拍摄。
- **优点**:界面直观、上手快;数据集模式让新手**无需排程即可拿到练手数据**;设备口径跨度大、双半球。
- **缺点**:点数制下高强度使用需较高订阅档;高级请求受排队与天气影响。

**适合**:既想快速拿现成数据练后期、又希望逐步过渡到自定义拍摄的人。

## Slooh

- **类别**:面向**教育与新手**的在线望远镜平台与社区,强调浏览器内「实况观看」与天象直播。创立于 2002 年。
- **设备 / 台址**:**西班牙特内里费泰德山(Mount Teide,约 2300m)**与**智利**(以及澳大利亚 Siding Spring)等台址,共数台望远镜(含高倍 14 英寸施密特-卡塞格林与广角镜)。
- **计费(分级会员)**:面向个人 / 学生 / 家庭 / 教育机构的分层订阅(常见两档:入门档「Apprentice」可观测约 500 个热门目标;进阶档「Astronomer」可观测更冷门目标)。
- **使用模式**:在线排队预约望远镜进行个人观测,平台并组织日食、月食、彗星等**天象直播**;有完善的课程与学习资源。
- **优点**:门槛极低、强社区与教育属性、可参与实时天象活动;南北半球台址兼顾。
- **缺点**:设备规模与可定制性有限,**不适合追求高画质深空成像**或科研级数据的进阶用户。

**适合**:纯新手、课堂教学、对「看见 + 参与」比「极致出片」更看重的人。

## ChileScope

- **类别**:智利**大口径**远程望远镜服务,主打优良视宁度。
- **设备 / 台址**:位于智利安第斯 / 阿塔卡马南缘,海拔约 **1560m**,距 Cerro Pachón 约 20 公里。三台主力设备:**1 米 f/8 Ritchey–Chrétien**(Alt-Az 直驱)与**两台 0.5 米(500mm)f/3.8 牛顿**(赤道仪),配 FLI ProLine 16803 大画幅相机与 8–10 位滤镜轮。
- **视宁度**:平均**优于 1 角秒**,好夜约 **0.6–0.8 角秒**,属业余可及的高分辨率台址。
- **计费(点数制)**:以点数计费(约 **1 点 ≈ 1 美元**),按所选望远镜的「点 / 帧」费率实时扣除;通过 Web 排程系统自定义目标、相机、滤镜、曝光,数据自动下传。
- **优点**:**大口径 + 亚角秒视宁**,适合小星系、星系细节、暗弱目标与高分辨率窄带;南天台址。
- **缺点**:大镜机时单价高,深度成像花费可观;面向有经验的成像者,排程参数需自行掌握。

**适合**:追求高分辨率、暗弱小目标、愿为优质台址付费的进阶 / 高级成像者。

## Deep Sky Chile

- **类别**:智利远程**托管(hosting)**与远程运营服务——核心是把**你自己的设备**架到对方台址,而非按夜租用对方的镜。
- **设备 / 台址**:位于智利 **Río Hurtado 河谷**一处平顶山,海拔约 **1700m**,光害极低,平均**视宁度优于 1 角秒**;晴夜率在智利乃至全球名列前茅。自 2019 年提供托管。
- **基础设施**:全自动卷帘顶 / 圆顶观测站,太阳能供电,约 **100Mbps 低延迟**网络(40Mbps 4G 备份),含安防与告警系统;可选入驻共享卷帘顶观测站或自建私人观测站。
- **计费**:按托管(机位 / 圆顶租赁 + 运营)收费,而非点数 / 机时;具体随设备体量与方案而定。
- **优点**:把自有设备「移民」到南天暗空,长期使用边际成本低、完全自有控制;台址条件顶级。
- **缺点**:**需自购并托运设备**、有前期投入与运维责任,门槛高于「按机时租镜」;不适合只想偶尔拍几张的用户。

**适合**:已有成熟成像系统、希望长期获得南半球暗空与稳定运营的资深玩家或团队。

## Insight Observatory

- **类别**:**教育导向**的远程望远镜与图片服务,运营 ATEO(Astronomical Telescopes for Educational Outreach)门户。
- **设备 / 台址**:自 2017 年起整合 **15+ 联盟远程望远镜**进 ATEO 门户,台址分布于新墨西哥、中国丽江等多地。
- **计费(按请求 / 按分钟)**:提供 Basic / Advanced / Educational 三类「图像请求」,按**成像分钟**计费(教育价示例:约 $0.36–0.65 / 分钟,处理后成片约 $25–50);另有订阅式图库「Starbase」可购买下载高质量成片。
- **使用模式**:以「提交目标请求 → 平台代拍 → 交付数据 / 成片」为主,面向 K-12 课堂、图书馆等公共教育场景。
- **优点**:对教师 / 学生友好,有教育定价与课堂图像请求(EIR);多台址可选。
- **缺点**:更偏教育与代拍,实时控制与高端定制弱于专业网络平台。

**适合**:学校、教育机构、希望以低成本完成科普 / 教学成像任务的用户。

## 选用建议

平台选择取决于「你处于哪个阶段、要拍什么、预算与精力如何」。一个由浅入深的顺序:

1. **查参数(任何阶段)**:用 **AstroBin** 检索目标与相近焦距的样张,参考曝光、滤镜与构图,并把成果归档。
2. **零基础体验**:用 **Slooh** 或 **Telescope Live** 的 One-Click 数据集,先在低门槛下「看见」并拿到现成数据练[后期叠加](/astrophotography/processing/stacking/)。
3. **第一次自己拍广角深空**:用 **SkyShare Astro** 这类按夜平台,流程简单、有天气保证,直接拿 FITS。
4. **需要更多设备与半球**:用 **iTelescope** 或 **Telescope Live** 的高级请求,选更长焦距、不同半球或特殊滤镜配置;科研需求优先 iTelescope。
5. **追求高分辨率小目标**:用 **ChileScope** 等大口径 + 亚角秒视宁的台址。
6. **长期 / 拥有自有设备**:用 **Deep Sky Chile** 等托管服务,把设备架到南天暗空长期运营。
7. **教育 / 课堂**:用 **Insight Observatory** 或 **Slooh** 的教育方案。

<Aside type="tip" title="区分用途">
**拍摄获取数据**靠 iTelescope / SkyShare / Telescope Live / ChileScope 等;**存图、展示、查参数**靠 AstroBin。两者通常配合使用:先在 AstroBin 查目标参数,再到远程平台获取 FITS,处理后回 AstroBin 归档。
</Aside>

下一步可阅读[远程成像入门](/astrophotography/remote-platforms/overview/)了解流程,把拿到的 FITS 数据接入[后期叠加](/astrophotography/processing/stacking/),并参考[天体星表](/reference/catalog/)挑选适合各平台视场的目标。
